## 우선 검토

- `table_index 204`
  - `입찰공고번호 / 입찰일자 / 입찰건명`
  - 필드형 표에 가깝고, 입찰 메타데이터라 RAG 가치가 있음
  - 현재는 헤더/필드 그룹 매칭이 약해서 버려짐

- `table_index 21`
  - `스톡옵션 통계 / 복수의결권 통계 / 벤처확인 + 스톡옵션 ...`
  - 통계/메뉴맵형 구조일 가능성이 있음
  - 일반 헤더형 표는 아니지만 시스템 기능/통계 축으로 중요할 수 있음

- `table_index 18`
  - `복수의결권 / 스톡옵션 개념도 / 메뉴 구성(안)`
  - 도식/메뉴 구성 시작 블록일 가능성이 있음
  - 단독 표라기보다 이미지/도식과 연결되는 제목 블록일 수 있음

- `table_index 19`
  - `대민 / 내부 BackOffice`
  - 좌측/우측 레이블형 구조 조각일 가능성이 있음
  - 후속 블록과 연결되는지 확인 필요

- `table_index 40`
  - `제안 요구사항 / 요구사항 정의표`
  - 실제 요구사항 표의 제목 블록일 가능성이 큼
  - 다음 표와 연결 규칙이 필요할 수 있음

- `table_index 42`
  - `기능 요구사항(System Function Requirement)`
  - 요구사항 묶음의 카테고리 블록
  - 제목 블록이지만 후속 표 검색 힌트로는 중요함
